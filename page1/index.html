ev4解密为MP4(无视授权密码)

大家下载教程的时候很多教程都是ev4格式的视频，这种视频是需要授权密码才能播放的。

遇到这种情况，可以用本程序，可以把 .ev4格式的视频转换为 .mp4格式的。

这样就可以用任意播放器播放该视频了，最重要的是无须知道授权密码也可以转换哦！

下面介绍软件的使用教程：

1.下载软件后 打开文件夹 ，在文件夹里面打开 cmd 。如图：

2.在 cmd 中里面输入 main 然后加上一个空格 最后拖动需要转换的.ev4格式的加密视频到cmd中。

'     注意1：需要转换的视频路径中不能有空格。

'     注意2：路径一定要有引号，直接拖动视频到cmd中路径会自带引号。

3.输入完毕后， 按下回车即可。可以看到效果如下：
